CASE 1.Регистрация на сайте https://qa-complex-app-for-testing.herokuapp.com/
Steps:
1. Заполнить поле "Username" - имя пользователя "user00001"
2. Заполнить поле "Email" - почтовый адрес "user00001@gmail.com"
3. Заполнить поле "Password" - пароль "123456qwerty"
4. Нажать кнопку "Sign up for OurApp"

Actual result:
1. В header отображено имя пользователя под которым выполнена регистрация на сайте
2. Отображено приветственное сообщение: Hello %USERNAME%, your feed is empty.



CASE 2. Выход из профиля пользователя - Sign Out
Precondition:
1. Залогинится на сайте https://qa-complex-app-for-testing.herokuapp.com/ под только что созаднным пользователем
Steps:
1. Нажать в header кнопку "Sign Out"

Actual result:
1. В header отбражена кнопка "Sign In"
2. Отображены поля для входа в профиль пользователя на сайте - "Username" и "Password"
3. Отображены поля для регистрации пользователя - "Username", "Email" и "Password"



CASE 3. Вход в профиль на сайте зарегистрированным пользователем
Precondition:
1. Залогинится на сайте https://qa-complex-app-for-testing.herokuapp.com/ под только что созаднным пользователем
Steps:
1. В поле "Username" указать имя пользователя под которым он зарегистрирован на сайте
2. В поле "Password" указать пароль этого пользователя
3. Нажать кнопку "Sign In"

Actual result:
1. В header отображена кнопка "Create Post"
2. В header отбражена кнопка "Sign Out"



CASE 4. Создание сообщения - Create Post
Precondition:
1. Залогинится на сайте https://qa-complex-app-for-testing.herokuapp.com/ под только что созаднным пользователем

1. Нажать кнопку "Create Post"
2. Заполнить название сообщения в поле Title - "Это мой первый пост"
3. Заполнить поле "Body Content" текстом сообщения:
"Это мой первый пост на этом сайте"
4. Установить флаг "This is a unique post"
5. Выбрать значение выпадающего списка "Select value" - Общедоступное
6. Нажать кнопку "Save New Post"

Actual result:
1. На зелёном фоне отображено сообщение сайта - "New post successfully created."
2. Отображено название только что созданного сообщения - "Это мой первый пост"
3. Отбражен сообщение сайта "Note: This post was written for All Users"
4. Отображен текст сообщения "Это мой первый пост на этом сайте"



CASE 5. Редактирование только что созданного сообщения - Edit
Precondition:
1. Залогинится на сайте https://qa-complex-app-for-testing.herokuapp.com/ под только что созаднным пользователем
2. Создать новое сообщение

Steps:
1. Перейти в профиль - синяя кнопка "My Profile"
2. На закладке "Posts:" нажать строку с названием "Это мой первый пост"
3. В открывшемся сообщении нажать кнопку "Edit" (листочек с карандашом)
4. В поле "Body Content" добавить в конце текста восклицательный знак
5. Установить флаг "This is a unique post"
6. Нажать "Save Updates"

Actual result:
1. На зелёном поле появилось сообщение "Post successfully updated."
2. Обратно перейти через профиль (шаги 1-2) в только что отредактирвоаное сообщение.
3. Текст сообщения теперь содержит "!" - "Это мой первый пост на этом сайте!"
